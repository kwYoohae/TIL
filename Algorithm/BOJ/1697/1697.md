# [BOJ] 1697 숨바꼭질
![문제](https://user-images.githubusercontent.com/74089271/147209742-15c558e0-e21d-4179-9d7f-97419a492508.png)

### 문제풀이
이 문제를 풀기 위해서 DataStruct강의 중 풀었던 BFS로 문제를 풀었습니다.

코드의 최단 경로의 거치는 Path의 수를 찾으면 되기 때문에 이를 얻기 위해서 `Stack`을 사용했습니다. 

`Stack`에 BFS로 거치는 경우의 수를 `Pair` Class를 만들어서 작성하였습니다.


문제를 풀 때 방문을 확인하는 `Visited`의 범위를 넘지 않게 if문으로 넘는지 않넘는지 확인을 해주었습니다. (음수인 경우, 양수인 경우)


### 정답 코드
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.Queue;
import java.util.Stack;

class Pair { // Pair 클래스 생성
    int first;
    int second;
    Pair(int first, int second){
        super();
        this.first = first;
        this.second = second;
    }
   
}

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        ArrayList<Integer> array = new ArrayList<>();
        Queue<Integer> queue = new LinkedList<Integer>();
        Stack<Pair> path = new Stack<>();
        boolean[] visited = new boolean[100001];


        int count = 0;
        int endNumber = 0;

        String[] read = br.readLine().split(" ");
        int start = Integer.parseInt(read[0]);
        int end = Integer.parseInt(read[1]);

        queue.add(start); // Queue에 데이터 넣기
        while(!queue.isEmpty()){
            int front = queue.poll(); // Front에 데이터 복사 후 삭제
            visited[front] = true;
            if(front+1 <= 100000){ // 범위를 넘는지 확인
                if(visited[front+1] == false){
                    queue.add(front+1);
                    path.add(new Pair(front + 1, front));
                    visited[front+1] = true;
                }
            }
            if(front-1 >= 0){ // 범위가 음수가 되는지 확인
                if(visited[front-1] == false){
                    queue.add(front-1);
                    path.add(new Pair(front - 1, front));
                    visited[front-1] = true;
                }
            }
            if(front*2 <= 100000){ // 범위를 넘는지 확인
                if(visited[front*2] == false){
                    queue.add(front * 2);
                    path.add(new Pair(front*2, front));
                    visited[front*2] = true;
                }
            }
            if(front+1 == end || front-1 == end || front*2 == end){ // 만약 도착점에 도착을 하면 반복문 종료
                break;
            }
        }

        while(!path.empty()) // Path경로를 역추적
        {
            if(path.peek().first == end){ // 값을 찾았으면 이전 값을 path에서 가져와서 다시 찾는다. 
                end = path.peek().second;
                count++;
            }
            path.pop();
        }

        System.out.println(count);
    }
}
```
