# [BOJ] 2579 계단오르기
![문제](https://user-images.githubusercontent.com/74089271/147534597-9630f760-257c-4a85-ad32-815c30ed76f0.png)

### 문제풀이
이 문제는 Dynamic Programing 문제입니다.

우선 규칙을 잘 모르겠어서 규칙을 찾아보고 다음과 같은 과정을 통해 점화식을 찾았습니다.

![풀이](https://user-images.githubusercontent.com/74089271/147534734-3d3c996e-3962-48ac-907f-9b44cfdedbd3.png)

단순한 규칙으로 2칸 전의 값과 1칸 전의 값을 보고 비교해나가면 됩니다.

2칸 전의 값은 Count값에 상관없이 어느거나 고를 수 있기 때문에 2칸전의 값중 가장 큰 값을 고르면 됩니다.   


하지만 한 칸전의 값은 Count가 2인 값을 고르면 규칙에 위반되기 때문에 Count가 1인 dp[i-1][1]의 값만 고를 수 있습니다.

결론적으로는 다음과 같이 됩니다.
> `dp[i-1][1] + arr[i]`  
`Max(dp[i-2][0] , dp[i-2][1]) + arr[i]`

### 정답 코드
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int number = Integer.parseInt(br.readLine());

        int[] arr = new int[number];
        int[][] dp = new int[number][2];

        for(int i = 0; i < number; i++){
            arr[i] = Integer.parseInt(br.readLine());
        }
        //처음 값은 첫 번째 값으로 초기화
        dp[0][0] = arr[0];
        dp[0][1] = arr[0]; 


        for(int i = 1 ; i < number; i++){
            if(i == 1){ // 두 번째 값은 i-2가 없기 때문에 예외처리 해준다.
                dp[i][0] = dp[i-1][1] + arr[i];
                dp[i][1] = arr[i];
            }else{
                // 우리가 구한 점화식을 통해서 값을 넣어준다.
                dp[i][0] = dp[i-1][1] + arr[i];
                dp[i][1] = Math.max(dp[i-2][0], dp[i-2][1]) + arr[i];
            }
        }

        // 가장 큰 값을 출력
        System.out.println(Math.max(dp[number-1][0], dp[number-1][1]));
    }
}

```
